# Apex

A CLI scaffolder for Go projects with selectable architectures.

## Install (Go)
```
go install github.com/<bruncdev>/apex@latest
```

## Install (Binary Release)
Download a prebuilt binary from the GitHub Releases page.

## Usage
```
apex new
```

## Release (maintainers)
Create a tag to publish a release:
```
git tag v0.1.0
git push origin v0.1.0
```
